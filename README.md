# EcoVolt Insights ⚡🌱

EcoVolt Insights is a web-based dashboard focused on waste-to-energy analysis and sustainability insights.  
The application helps users understand how different types of waste can be converted into usable energy through clear visualizations and step-by-step explanations.

---

##  Features

- Interactive sustainability dashboard
- Waste-to-energy process identification
- Detailed step-by-step energy generation explanation
- Clean and minimal UI design
- Responsive layout for all screen sizes

---

##  Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

---

## Getting Started

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd ecovolt-insights

# Install dependencies
npm install

# Start the development server
npm run dev
